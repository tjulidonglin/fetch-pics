=======================================
  🎯 部署步骤概览
=======================================

1. 安装依赖: pip3 install -r requirements.txt
2. 配置 Nginx: 复制配置文件并启用
3. 设置服务: 配置 systemd 服务
4. 启动服务: systemctl start snake_game
5. 访问游戏: 打开浏览器访问 http://localhost

（详细步骤请查看 deploy_snake_game.sh）
